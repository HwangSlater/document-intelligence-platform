# 프로젝트명

Document Intelligence Platform

## 부제

Embabel 기반 Agentic Document Intelligence System

---

# 프로젝트 개요

Document Intelligence Platform은 생성형 AI, RAG, Agent Workflow를 활용하여 문서를 이해하고 활용하는 업무 지원형 AI 플랫폼이다.

단순한 문서 QA 챗봇이 아니라 문서를 분석하고, 정보를 추출하고, 비교하며, 사용자의 업무를 지원하는 것을 목표로 한다.

---

# 프로젝트 목표

사용자는 문서를 읽고 싶어서 시스템을 사용하는 것이 아니다.

사용자는 업무를 해결하기 위해 시스템을 사용한다.

따라서 시스템은 단순 검색 기능을 넘어서 다음 역할을 수행해야 한다.

* 문서 탐색
* 정보 검색
* 문서 요약
* 정보 추출
* 문서 비교
* 업무 지원

---

# 핵심 가치

기존

문서 → 사용자가 읽음

개선

문서 → AI가 이해 → 사용자에게 결과 제공

---

# 주요 기능

* Document Search

# 기대 효과
