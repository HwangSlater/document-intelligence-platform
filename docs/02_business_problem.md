# 문제 정의

조직 내에는 수많은 문서가 존재한다.

예시

* 업무 매뉴얼
* 규정집
* 정책 문서
* 회의록
* 기술 문서
* API 문서

하지만 실제 사용자는 필요한 정보를 찾기 위해 문서를 직접 탐색해야 한다.
